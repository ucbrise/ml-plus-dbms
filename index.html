ML-powered Database Systems

What is the role of machine learning in the design and implementation of a modern database system? Advances in deep learning have achieved "super-human" performances in various domains due to the confluence of specialized compute, abundant data, and expressive architectures. In this project, we investigate how machine learning can help improve or design better data systems.

Deep Query Optimization

An emerging body of work has shown that there is tremendous potential upside in applying deep learning to query optimization. In our recent work, we built DQ, a select-project-join optimizer that chooses an optimal join ordering for a specific dataset, workload, and hardware combination, based on Q-learning ideas.

Ongoing work includes extending DQ to new types of joins and enriching its action space.
